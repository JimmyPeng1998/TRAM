# TRAM
**T**ucker **r**ank-**a**daptive **m**ethod for Tucker tensor completion



## How to run?

- Run ``Run_Me_First.m`` 
- Run ``Test_increasing.m`` for testing the rank-increasing procedure
- Run ``Test_decreasing.m`` for testing the rank-decreasing procedure
